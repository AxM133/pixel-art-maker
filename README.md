### ‚ú® Features
- **Dynamic Canvas Size:** Choose presets (16x16, 32x32) or set a custom grid size (up to 64x64). Includes input validation.
- **Advanced Tools:**
  - ‚úèÔ∏è **Pencil:** Draw pixels precisely.
  - üßΩ **Eraser:** Smart eraser that clears pixels without resetting your selected color.
  - üñä **Eyedropper (Picker):** Quickly pick any color from the canvas.
  - üóë **Clear Canvas:** Reset the grid instantly.
- **Color System:**
  - Curated palette of essential colors.
  - **üåà Custom Color Picker:** Native integration to choose ANY color from the spectrum.
- **Smart Export:** Download artwork as high-quality `.png`. The app automatically upscales the image (x20) to prevent blurriness.
- **Modern UI:** Responsive design with dark mode and glassmorphism effects.

### üõ† Tech Stack

* **Styling:** CSS3 (Grid, Flexbox, Custom Properties)
